# `notebooklite`

each comes in two flavours, with or without a notebook name

+++

## nb no args


```{notebooklite}
:theme: JupyterLab Light
:width: 100%
:height: 500px
:prompt: notebooklite no args
:prompt_color: red
```

+++

## nb on hello-world.ipynb

omitting the theme

```{notebooklite} hello-world.ipynb
:width: 100%
:height: 500px
:prompt: notebooklite on one notebook
:prompt_color: orange
```

+++
